1)Рекомендуется использовать виртуальное окружение (venv)
2)Используется python 3.12
3)установка зависимостей: pip install -r requirements.txt
4)запуск: uvicorn main:app --reload
5)в браузере http://127.0.0.1:8000/
6)авто документация http://127.0.0.1:8000/docs
7)запуск фронта -> templates -> index.html при включенном сервере
